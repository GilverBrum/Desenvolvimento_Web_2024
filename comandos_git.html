Seguem os comandos do git:

git clone -> clona o repositório do git para a máquina local

git add -A -> adiciona todos os arquivos do diretório para serem gerenciados pelo git

git commit -m "mensagem do comit que vc quiser" -> cria um pacote com as alterações realizadas (criação de arquivos, remoção de arquivos, alteraçãoem arquivos, etc)

git push -> envia os commits para o repositório remoto (no nosso caso , o github)

git pull -> traz as alterações do repositórioremoto para o repositório local
